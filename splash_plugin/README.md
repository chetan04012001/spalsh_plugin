## splash_plugin
 A Flutter plugin for creating Splash Screen.

## Features
Instant splash Screen Ready: 
just add :
1) Route 
2) Image 
3) Welcome Text
4) Duration

## Getting started
To use this package add splash_plugin as a dependency in your pubspec.yaml
